Help & Using IRD-GenDB

Welcome to IRD-GenDB

IRD-GenDB is a genetic variant database for Inherited Retinal Diseases (IRD). The database stores patient data, genetic variants, and associated information to help researchers and clinicians manage and search IRD-related genetic data.

Dashboard

The Dashboard provides a quick overview of the database contents:

  • Total Patients: Number of patients in the database
  • Total Variants: Number of recorded genetic variants
  • Unique Genes: Number of distinct genes with variants
  • Most Common Gene: Gene with the highest number of variants

Tip Use the Dashboard as your quick reference for database statistics.

Patients Page

The Patients page displays a browsable list of all patients in the database and their corresponding age, diagnosis, inheritance, and gene information. Use the filters to narrow down results:

Filter Fields:
Field Input Type Description
Diagnosis Dropdown Select a diagnosis category (e.g., Leber Congenital Amaurosis, Retinitis Pigmentosa)
Sex Dropdown Filter by Male (M) or Female (F)
Inheritance Dropdown Select inheritance pattern (e.g., Autosomal Dominant, Autosomal Recessive, X-Linked)
Gene Text Input Enter gene name (e.g., CEP290, RHO, USH2A)
Actions:
  • Click Filter button (funnel icon) to apply filters
  • Click Clear button (X icon) to reset all filters
  • Click Export CSV to download filtered patient data
Variants Page

The Variants page shows genetic variants with two main features:

Variant Hotspots (Tabs):
  • By Diagnosis: Select a diagnosis group to see the most frequently observed variants for that condition
  • By Gene: Select a gene to view variants commonly found in that gene
Variant Table Columns:
Column Description
Sample ID Unique identifier for the patient sample
Gene Gene name associated with the variant (e.g., CEP290, RHO)
HGVS_c. cDNA notation describing the nucleotide change (e.g., c.5658C>T)
Protein Change Amino acid change notation (e.g., p.Gly1886Asp)
Pathogenicity Classification: Pathogenic, Likely Pathogenic, Benign, etc.
Variant Type Type of mutation: SNV, Deletion, Insertion, etc.
Zygosity Homozygous or Heterozygous
Search Page

The Search page provides two search modes:

Search Modes:
Mode Input Format What It Finds
Search by Gene Text (e.g., CEP290) All variants and patients associated with a specific gene
Search by Diagnosis Text (e.g., RP, LCA) All variants and patients with a specific diagnosis
How to Search:
  1. Select a search mode by clicking the corresponding button
  2. Enter your search term in the text box
  3. Click Search or press Enter
  4. Results will display in a table below
Autocomplete: As you type, suggestions may appear. Select from the dropdown to quickly fill in valid search terms.

Data Format Reference
HGVS Nomenclature:

Genetic variants are described using HGVS (HGAMG Variant Sequence) notation:

  • c. (coding DNA) - e.g., c.5658C>T means nucleotide 5658 changed from C to T
  • p. (protein) - e.g., p.Gly1886Asp means amino acid Gly changed to Asp at position 1886
Common Diagnosis Codes:
  • RP - Retinitis Pigmentosa
  • LCA - Leber Congenital Amaurosis
  • Stargardt - Stargardt Disease (ABCA4)
  • CRD - Cone-Rod Dystrophy
Common Inheritance Patterns:
  • AD - Autosomal Dominant
  • AR - Autosomal Recessive
  • XLR - X-Linked Recessive
  • Mitochondrial - Maternal inheritance
